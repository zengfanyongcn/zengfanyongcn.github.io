NexT 主题自定义侧边栏图标

发表于 2022-04-20 分类于 code ， NexT 阅读次数： 本文字数： 1.4k 阅读时长 ≈ 1 分钟

NexT 主题的图标基本上都是由 Font Awesome 提供，但是知乎、CSDN、bilibili等大多数国内应用软件的图标在 Font Awesome 都不支持。为了支持侧边栏各种应用小图标的显示，可以利用嵌入 svg 格式的图标进行实现。

下载 icon 图标

阿里的 iconfont 网站上拥有丰富的图标，通过这个网站下载 svg 格式的图标，并将图片保存在 Hexo 文件下的 source/images 路径下，如果 images 文件不存在，则新建 images 文件夹。（注意：这里是 Hexo下的文件夹下，不是主题文件 NexT 的路径下）

设置图标格式
